# Section 7.4 OpenVPN

OpenVPN is available in the package.

# Install OpenVPN

- Install with pkg:

```sh '
# Pkg install openvpn
````

- Or install with Ports:

````
#cd/usr/ports/security/openvpn/
# Make install clean
````
